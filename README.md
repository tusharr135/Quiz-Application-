# Quiz Application

A simple Quiz Application developed in Python that asks multiple-choice questions, checks answers, and displays the final score.

---

## Features

✅ Multiple Choice Questions

✅ Automatic Score Calculation

✅ Instant Feedback (Correct/Wrong)

✅ Final Result Display

✅ Beginner-Friendly Python Project

---

## Technologies Used

- Python 3

## How It Works

1. The program displays a question with options.
2. The user enters an answer.
3. The program checks whether the answer is correct.
4. Score is updated automatically.
5. After all questions, the final score is displayed.
